IntelliJ Plugin

The latest version of the IntelliJ plugin is available at the JetBrains Marketplace. It can be installed directly from the IDE, see the IntelliJ IDEA documentation for details.

Previous releases are available at the DataSonnet plugin Github repository.

Installing the plugin from a binary distribution file.

  • Download the plugin archive (ZIP or JAR) from the Releases page.

  • Press ⌘ Comma to open the IDE settings and select Plugins.

  • On the Plugins page, click the Settings button and then click Install Plugin from Disk….

  • Select the plugin archive file and click OK.

  • Click OK to apply the changes and restart the IDE if prompted.

See the IntelliJ IDEA documentation for more information.
